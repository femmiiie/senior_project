# ipass

Umbrella header that includes the entire library.

## Includes

- [`PatchData.h`](/components/PatchData): Core data types, configuration, and status codes.
- [`LODPass.h`](/components/LODPass): Standalone LOD compute pass.
- [`TessellationPass.h`](/components/TessellationPass): Standalone tessellation compute pass.
- [`Pipeline.h`](/components/Pipeline): High-level pipeline combining LOD and tessellation.
- [`BVLoader.h`](/components/BVLoader): Utility for loading `.bv` patch files.
